<h2 id=“はじめに”>はじめに

最初は Ruby の勉強のために始めた競技プログラミング（以下競プロ）ですが、最近はほぼ趣味として取り組んでいます。いずれは競プロに適しているとされる、コンパイル言語の C++で取り組んでみたいとは思っているものの、まだ Ruby 学習中の身なのでしばらくは Ruby でやっていこうと思っています。

さて、僕の競プロ歴は AtCoder の ABC に 4 回出場、実力は 3 完できるくらいの初心者ではありますが、Ruby で競プロを行う上で役に立つ Tips をブログに書き溜めていきたいと思い立ち、この記事を書いています。

競プロの Ruby 情報は多くないと思いますので、誰かの役に立てれば幸いです。

ということで、今回は競プロでは必須の「標準入力を扱う方法」についてです。

<h2 id=“gets メソッドで標準入力を取得できる”>gets メソッドで標準入力を取得できる

Ruby では、<code>gets</code>で標準入力を取得できます。ただし、<code>gets</code>で取得した文字列は改行コード付きのため、競プロにおいては<code>gets</code>単独ではほとんど使いません。

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ sample</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets=&gt; <span class=“synSpecial”>&quot;<span class=“synConstant”>sample<span class=“synSpecial”>\n&quot; <span class=“synComment”># 末尾に改行コード</pre>

<h2 id=“文字列を扱う場合”>文字列を扱う場合

入力を文字列で扱う場合は、<code>chomp</code>を使って末尾の改行コードを取り除きます。

これをしないと、パッと見正しい出力ができていても、オンラインジャッジで誤答と判定される場合があるので注意です。このことを知らず、「明らかに正しい出力が得られているのに AC にならない。なぜ。。。」とドツボにはまったことがありました。

ちなみに、入力を数値で扱う場合は<code>chomp</code>は必要ありません。<code>chomp</code>を付けると処理速度の低下につながるようなので、文字列を扱う場合のみ付けるようにしましょう。

＜参考＞

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ sample</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.chomp =&gt; <span class=“synSpecial”>&quot;<span class=“synConstant”>sample<span class=“synSpecial”>&quot;</pre>

<h3 id=“文字列を 1 文字ずつ分けて配列に格納する”>文字列を 1 文字ずつ分けて配列に格納する

入力の文字列を、1 文字ずつ扱いたい場合は<code>chars</code>を使って分割し配列に格納できます。

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ sample</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.chomp.chars =&gt; [<span class=“synSpecial”>&quot;<span class=“synConstant”>s<span class=“synSpecial”>&quot;, <span class=“synSpecial”>&quot;<span class=“synConstant”>a<span class=“synSpecial”>&quot;, <span class=“synSpecial”>&quot;<span class=“synConstant”>m<span class=“synSpecial”>&quot;, <span class=“synSpecial”>&quot;<span class=“synConstant”>p<span class=“synSpecial”>&quot;, <span class=“synSpecial”>&quot;<span class=“synConstant”>l<span class=“synSpecial”>&quot;, <span class=“synSpecial”>&quot;<span class=“synConstant”>e<span class=“synSpecial”>&quot;]</pre>

<h2 id=“数値を扱う場合”>数値を扱う場合

<h3 id=“整数として取得する”>整数として取得する

<code>gets</code>は文字列を返すので、数値として扱いたい場合は<code>to_i</code>で整数に変換します。

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ 32</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.to_i =&gt; <span class=“synConstant”>32</pre>

数値を文字列として扱いたい場合は<code>gets.chomp</code>を使います。

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.chomp=&gt; <span class=“synSpecial”>&quot;<span class=“synConstant”>32<span class=“synSpecial”>&quot;</pre>

<h3 id=“浮動小数点数として取得する”>浮動小数点数として取得する

入力値を小数で扱いたい場合は<code>to_f</code>を使います。

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ 32</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.to_f=&gt; <span class=“synConstant”>32.0</pre>

<h3 id=“スペース区切りの数値を取得する”>スペース区切りの数値を取得する

競プロでよくあるのがスペース区切りで数値が並んでいる入力です。この場合は、<code>split</code>を使って各数値を分割します。

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ 5 3 12 1</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.split.map(&amp;<span class=“synConstant”>:to_i）=&gt; [<span class=“synConstant”>5, <span class=“synConstant”>3, <span class=“synConstant”>12, <span class=“synConstant”>1]</pre>

たまに<code>split(' ‘)</code>と引数に区切り文字の半角スペースを指定している人を見かけますが、<code>split</code>は引数を省略すると半角スペースで区切ってくれるので、結果は変わりません。

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>x = gets.split(<span class=“synSpecial”>'<span class=“synConstant”> <span class=“synSpecial”>').map(&amp;<span class=“synConstant”>:to_i）=&gt; [<span class=“synConstant”>5, <span class=“synConstant”>3, <span class=“synConstant”>12, <span class=“synConstant”>1]</pre>

各数値を配列でなく、独立した変数に格納したい場合は多重代入を使います。

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>a, b, c, d = gets.split.map(&amp;<span class=“synConstant”>:to_i）=&gt; [<span class=“synConstant”>5, <span class=“synConstant”>3, <span class=“synConstant”>12, <span class=“synConstant”>1]

p a=&gt; <span class=“synConstant”>5

p c=&gt; <span class=“synConstant”>12</pre>

<h3 id=“縦に並んだ数値を取得する”>縦に並んだ数値を取得する

縦に数値がならんだ入力を扱う方法です。

最初に「数値の数」が入力として与えられ、以降、縦に数値が並ぶケースが多いです。

以下の例だと、数値の数「4」が与えられ、以降、4 つの数値が縦に並びます。この場合は、4 回<code>gets.to_i</code>を繰り返せば OK です。

<pre class=“code” data-lang="" data-unlink>＜標準入力＞ 453121</pre>

<pre class=“code lang-ruby” data-lang=“ruby” data-unlink>n = gets.to_i=&gt; <span class=“synConstant”>4

n.times.map { gets.to_i } =&gt; [<span class=“synConstant”>5, <span class=“synConstant”>3, <span class=“synConstant”>12, <span class=“synConstant”>1]</pre>
