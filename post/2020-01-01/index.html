2019年を振り返る

2020/01/01
Dialy

2019 年はフリーランスになって 2 年目の年。まるまる 1 年フリーランスとして働いたはじめての年でもある。

ちょうどこの年末で別のプロジェクトに移ることになったのだが、SIer 出身で Java を扱ってきたこともあり、SIer さんからのオファーが多い。SIer さんの仕事は、古い技術が採用されていることが多かったり、リモートワークやフレックス制、ビジネスカジュアルなどといった柔軟な働き方ができる環境が整備されていることも稀だったりして、正直あまり魅力を感じない。

2020 年はよりモダンな労働環境を目指して精進する次第。

個人開発 Rails の学習を目的に、いくつかのアプリを開発した。

夏に Rails と Vue.js で実装した『ポケモンしりとり』をリリース。リリース当初はほとんど利用者がいなかったが、現在は 1 日に 100 人ほどの方が遊んでくれている。ありがとうございます ☺ 時間ができたら、ランキング機能を実装したいと思っている。（いつになることやら……。）

『ポケモンしりとり』の開発の過程では Rails の ActiveModel について深く学べなかったので、ゲームソフトを管理できるアプリケーションも開発した。読書管理サービスの『ブクログ』のゲーム版のようなイメージ。この開発によって学んだことは多かったが、いろんな意味でイマイチな点が多く、いまのところサービスとして正式リリースはしない予定。

読んだ本技術書を約 20 冊読んだ。Rails 関連、テスト技法、オブジェクト指向設計など。

その結果、技術書を読むことが得意じゃないことに気づく。読みやすくて上手く自分にハマれば苦なく読めるんだけど・・・。通勤中の電車は他にやることがないので、集中して読むことができた。読書を習慣にするには、環境つくりから工夫する必要がありそう。

ブログ 2019 年は 3 月から更新を始めて、56 記事書いた。毎週 1 回以上の更新することを目標に掲げ、達成できた。偉い。

一応、技術ブログのつもりだけど、技術記事、雑記、趣味などいろいろ書いた。今後は技術記事の比重を増やしていきたいなあ、とはなんとなく思っている。

「自分が読みたい記事だけ書いていく」方針はこれからも変わらず。

競技プログラミング夏頃から Ruby の勉強目的で始めた競プロ。単純に楽しくなってアルゴリズム本を読んだり本格的に取り組んでいたが、現在は他のスキルアップを優先するため休止中。
