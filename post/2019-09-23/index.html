【移行記事】iOS13リリース！便利な入力補助機能が増えたぞ

2019/09/23
Dialy

10:19:42

待ちに待った iOS13 がリリースされました！

なんといっても iOS13 の目玉機能は「ダークモード」！娘が寝ている近くでスマホを操作する機会が多いので、明るくなりすぎないダークモードの実装は非常にうれしい！

視覚的にも分かりやすいダークモードが注目されがちですが、iOS13 では、地味ながらとても役に立つ入力補助機能もいくつか追加されています。

僕はスマホでソースコードをメモしたり、ブログを書いたりする機会が多いので、僕のようなプログラマでブログを書いている人には特に便利だと思います。

iOS13 で登場した便利な入力補助機能

なぞり入力

<blockquote class=“twitter-tweet” data-lang=“ja”><p lang=“ja” dir=“ltr”>なぞり入力ができるように。アルファベット入力に便利。<a href=“https://twitter.com/hashtag/iOS13?src=hash&amp;amp;ref_src=twsrc%5Etfw"&gt;#iOS13&lt;/a> <a href=“https://t.co/G20gRoYRe0”>pic.twitter.com/G20gRoYRe0</a>&mdash; たてひと (@khattori89) <a href=“https://twitter.com/khattori89/status/1175167931031777281?ref_src=twsrc%5Etfw"&gt;2019年9月20日</a> <script async src=“https://platform.twitter.com/widgets.js" charset=“utf-8”></script>

英字キーボードをなぞって入力できるようになりました。ただ、単語を推測しているらしく、人名のような固有名詞は上手く入力できない（別の意図しない単語に置き換わってしまう）ことがあります。

ソースコードをささっとメモしたいときに便利そう！

快適なカーソル移動

<blockquote class=“twitter-tweet” data-lang=“ja”><p lang=“ja” dir=“ltr”>カーソルがぐにょぐにょ動くようになった。<a href=“https://twitter.com/hashtag/iOS13?src=hash&amp;amp;ref_src=twsrc%5Etfw"&gt;#iOS13&lt;/a> <a href=“https://t.co/T0cNZocOFa”>pic.twitter.com/T0cNZocOFa</a>&mdash; たてひと (@khattori89) <a href=“https://twitter.com/khattori89/status/1175168480414330880?ref_src=twsrc%5Etfw"&gt;2019年9月20日</a> <script async src=“https://platform.twitter.com/widgets.js" charset=“utf-8”></script>

以前よりカーソル移動が快適になり、狙った箇所にカーソルを移動させるのが楽になりました。

「取り消し」「やり直し」がジェスチャー操作可能に

<blockquote class=“twitter-tweet” data-lang=“ja”><p lang=“ja” dir=“ltr”>3 本指で左にスワイプで取り消し、右にスワイプでやり直し。<a href=“https://twitter.com/hashtag/iOS13?src=hash&amp;amp;ref_src=twsrc%5Etfw"&gt;#iOS13&lt;/a> <a href=“https://t.co/qZm8ydrcID”>pic.twitter.com/qZm8ydrcID</a>&mdash; たてひと (@khattori89) <a href=“https://twitter.com/khattori89/status/1175169082540187648?ref_src=twsrc%5Etfw"&gt;2019年9月20日</a> <script async src=“https://platform.twitter.com/widgets.js" charset=“utf-8”></script>

3 本指で左にスワイプで取り消し、同じく右にスワイプでやり直しができるようになりました。

これまでも iPhone を振ることで取り消しができましたが、振るよりも楽にできるようになった（人前で振るのは恥ずかしいし）のはうれしい。両手を使う必要はありますが。

コピー&amp;ペーストがジェスチャー操作可能に

<blockquote class=“twitter-tweet” data-lang=“ja”><p lang=“ja” dir=“ltr”>ピンチインでコピー、ピンチアウトでペースト。<a href=“https://twitter.com/hashtag/iOS13?src=hash&amp;amp;ref_src=twsrc%5Etfw"&gt;#iOS13&lt;/a> <a href=“https://t.co/roxTz0S5DW”>pic.twitter.com/roxTz0S5DW</a>&mdash; たてひと (@khattori89) <a href=“https://twitter.com/khattori89/status/1175167355732647936?ref_src=twsrc%5Etfw"&gt;2019年9月20日</a> <script async src=“https://platform.twitter.com/widgets.js" charset=“utf-8”></script>

同じくコピペも、3 本指ジャスチャーで操作可能になりました。3 本指でピンチインでコピー、ピンチアウトでペーストできます。

おわりに

他にも、

・音量表示が画面端へ・PS4 のコントローラー（DualShock4）が使用可能に・コントロールセンターで Wi-Fi と Bluetooth の接続先を切り替え可能に

などなど、iOS13 ではなかなかイケてる改善点が多い印象です！

ちなみに僕の iPhone7plus では、いまのところコレといったバグはないですが、Twitter などを見ていると、動作が不安定な端末もあるようです。不安な人は少し様子を見ても良いかもしれません。
