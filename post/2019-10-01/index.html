【移行記事】「iPhone7plus」から「iPhone11」に乗り換えたので、サイズ感とかカメラの使用感とか、感想を書く

2019/10/01
Dialy

08:00:00

3 年弱お世話になった「iPhone7plus」を手放し、「iPhone11」を購入しました！！

もともと購入する予定はなかったのですが、最近になって iPhone7plus のバッテリーの持ちが極端に悪くなったことがきっかけで、「せっかく iPhone11 が発売したんだからこれは買い替え時か～？」と iPhone11 の情報を調べていたらどんどん欲しくなってしまい、購入に至りました。妻も、iPhone8plus から iPhone11 に乗り換えました。出費……。（汗）

やっぱり新しい iPhone はいいですね。満足度は非常に高いです。

サイズ感

本体サイズ

iPhone7plus より縦幅が小さくなり、厚みは増しました。横幅はほぼ同じです。横幅がほぼ同じなので、持った感覚は、iPhone7plus とほとんど変わりません。片手持ちだと、端に親指が届きません。

片手で操作できないこともないけど、落としそうで怖いです（iPhone7plus を使っていたので、慣れてますが）。小さい iPhone を使っていた人は、慣れが必要です。（購入前に実機触ることをおすすめします）

カメラのレンズ部分は厚くなっているので、机などに置くと iPhone がガタついてしまいます。（これも今までの iPhone と同じ。）

本体サイズの体感は変わらないですが、ホームボタンがなくなったことによってディスプレイサイズは大きくなり、体感でかなり違います。最高。

純正クリアケース装着後

Apple 純正のクリアケースも購入しました。

純正クリアケースを装着すると、当たり前ですが、さらに厚みが増します。サイドの電源ボタンや音量ボタンは、結構強めに押さないと反応しません。

素材はかなりしっかりしていて、iPhone11 にバチッとぴったりハマります。さすがは純正。ぴったりすぎて、一度装着すると外すのに苦労するほどです。

ただ、背面に画像のような水たまりのような模様が出来てしまいました。これはおそらく iPhone11 とクリアケースの間に隙間がないためだと思われます。クリアケースを手で歪曲させてから装着すると良いかもしれません。

僕は純正が好きなので純正クリアケースにしましたが、妻は中華製の安いクリアケースを使っています。正直純正は価格が高いので、「純正が好き」「純正じゃなきゃいやだ」という人以外は、サードパーティ製のクリアケースで十分だと思います。

カメラ

超広角レンズ、ただただ撮るのが楽しいです。iPhone で遠近感のある写真を撮れることに感動。

ポートレート撮影も、大きくパワーアップしてます。ポートレート撮影は、被写体までの距離を一定に保たないと撮影できませんが、iPhone7plus より、距離感をそこまで気にしなくても撮影できるようになったように思います。（iPhone7plus のときは距離を保つのが難しく、ポートレート撮影はほとんど使わなくなってしまった）

我が家には娘という最高の被写体がいるので、iPhone での撮影、楽しいです。

OLYNPUS のミラーレス一眼も所有しているのですが、手軽さ、携帯性でいえば、iPhone11 の圧勝です。iPhone ではシャッタースピードや絞りを変えることはできませんが、（写真やカメラにそこまでこだわりがなく）そこそこのクオリティの写真を撮りたい、という人であれば、コスト面を込みで、iPhone11 で十分ではないでしょうか。（iPhone11pro なら望遠レンズもありますし。）

バッテリーの持ち

3 年弱使用した iPhone7plus と、新品の iPhone11 は単純比較はできませんが、純粋にバッテリーの持ちが良くなったのは嬉しいです。ちなみに、iPhone7plus は家を出るとき 100%だったのに、出勤したら 40%になってました。iPhone11 に替えたら、95%くらい残ってます。

その他

iPhone11 からの新機能ではないですが、iPhone7plus から乗り換えた身として、Face ID とホームバーについて触れておきます。

Touch ID（指紋認証）から Face ID（顔認証）に

Touch ID の人気が根強いと聞いていて、Touch ID がなくなることに不安がありましたが、Face ID は思ったより使いやすいです。iPhone11 から Face ID の認証速度が 30%アップしたらしく、認証が遅くてストレスを感じるようなことはありません。

特に、妻は指紋が薄く、Touch ID が反応しないと言っていたので、Face ID になって快適に使えるようになったみたいです。

ホームボタンからホームバーに

ホームボタンがなくなり、代わりに「ホームバー」になりました。慣れるまで少々時間はかかりましたが、特に不都合やストレスもなく使えています。ホームボタンがなくなったおかげで、ディスプレイサイズが大きくなったので、満足してます。

ただ、ちょっと気になるのが「スクリーンショットにホームバーが映りこんでしまう」ことです。少しググってみただけですが、どうやら設定で消すことはできないみたいで、現状は、後からトリミングなどするしかないようです。

おわりに

大満足です。新しい iPhone を手に入れると、それまで使っていた iPhone が急に古めかしく見えてしまいますね。iOS13 のアップデートもあり、iPhone をいじるのがすごく楽しいです。

ちなみに、「128GB の iPhone7plus」の Apple 公式下取り価格は約 22,640 円でした。iPhone11 は決して安くないですが（128GB は約 80,000 円）、下取りに出せば差額 57,000 円ほどで購入できます。SIM フリーの iPhone を購入するなら、Apple の下取りプログラムを利用するのが手軽でおすすめです！
