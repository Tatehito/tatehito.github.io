【移行記事】「Rails6でSemanticUIのJavaScriptを読み込む方法（ドロップダウン・アコーディオン等を使いたい！）」という記事をQiitaに投稿しました

2019/11/17
Tech Memo

23:07:31

<a href=“https://semantic-ui.com/"&gt;Semantic UI</a>はシンプルで洗練されたデザインが魅力的な CSS フレームワークです。以前は Bootstrap を利用していましたが、個人的にデザインが好みなのと、部品の種類が多いので、いま開発しているアプリでは Semantic UI を利用しています。

さて、Rails では<code>semantic-ui-scss</code>という gem を利用して Semantic UI をインストールすることができます。さらに Semantic UI の JavaScript を読み込めばドロップダウンなどの部品も使えるようになります。

しかし Rails6 から JQuery のインストール方法や application.js のパスが変わっているため、JavaScript を読み込む設定をするには少し手間がかかります。

ということで、Qiita に<a href=“https://qiita.com/tatehito/items/89dff4a42f0819f1371a">「Rails6でSemanticUIのJavaScriptを読み込む方法（ドロップダウン・アコーディオン等を使いたい！）」</a>という記事を書きました😃

僕と同じようにハマっている人が一定数いるのではないかと思い、今回は個人ブログより人目につきやすい Qiita に投稿してみました。

ドロップダウンが動かないよ〜という方の参考になれば幸いです！
